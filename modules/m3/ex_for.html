En for-loop går igenom en bestämd mängd element.
I detta fall talen 1 t o m 5.

      		
      
        
def ex1():
    print("The numbers 1-5")
    for num in [1,2,3,4,5]:
        print(num)

      		
    

      
        

          #
        

        
Elementen behöver inte vara tal, utan kan vara av vilken datatyp som helst. 

      		
      
        
def ex2a():
    print("A list of strings")
    for text in ['ABC', 'DEF', 'GHI']:
        print(text)

      		
    

      
        

          #
        

        
Elementen kan även vara av olika datatyp.

      		
      
        
def ex2b():
    print("A list of mixed items")
    for element in ['ABC', 10, 'DEF', 20.0, 'GHI']:
        print(element)

range

      		
      
        

      		
    

      
        

          #
        

        
Funktionen range kan användas för att skapa en lista av element.
Här skapas listan [0,1,2,3,4], vilket resulterar i att loopen körs 5 varv.

      		
      
        
def ex3a():
    print("The text 'Hello' five times")
    for i in range(5):
        print("Hello")

      		
    

      
        

          #
        

        
Man kan själv specificera startvärde om man inte vill ha 0. 

      		
      
        
def ex3b():
    for i in range(5, 10):
        print()i)

      		
    

      
        

          #
        

        
Man kan också ange andra inkrement (steg) än standardvärdet 1.
Här skapas listan [0,2,4,6,8]. För att utskriften ska bli lite roligare
så utförs också en liten beräkning.

      		
      
        
def ex4():
    print("Number\tSquare")
    for number in range(0, 10, 2):
        square = number * number
        print(number, "\t", square)

      		
    

      
        

          #
        

        
En for-loop går ett bestämt antal varv - men antalet skulle kunna bestämmas
av användaren.

      		
      
        
def ex5():
    print("THE REPEATER")
    name = input("Your name: ")
    num_iterations = input("How many times: ")
    for i in range(num_iterations):
        print(name, i)

"Ackumulator"

      		
      
        

      		
    

      
        

          #
        

        
En loop som summerar alla tal i ett intervall kallas för ackumulator.
Notera hur operatorn += används.

      		
      
        
def ex6():
    total = 0
    for number in range(10,21):
        total += number
    print("The sum of 10-20 is", total)

Nästlade loopar

      		
      
        

      		
    

      
        

          #
        

        
Man kan utföra det mesta inuti en loop - t ex en annan loop!
Detta kallas för att "nästla" loopar.

      		
      
        
def ex6a():
    for hour in range(24):
        for minute in range(60):
            print(hour, ':' , minute)

      		
    

      
        

          #
        

        
Här är ett exempel där antalet varv den inre loopen går, beror på vilket
varvet i den yttre loopen.

      		
      
        
def ex6b():
    for row_num in range(10, 30):
        for i in range(row_num):
            print(".",)
        print("")
    print("DONE")
